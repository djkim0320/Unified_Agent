# Research Mock Concept

Use this skill when the user asks for conceptual tradeoff tables, early design ranking, or a quick aerospace concept study.

## Tool routing

- Use `mock_concept.generate_concept_tradeoff` for a deterministic concept tradeoff run.

## Expected outputs

- `tradeoff.csv`
- `concept_report.md`

## Guidance

- Prefer this add-on when the request is about early sizing, comparative options, or ranked concepts.
- Keep the report short, structured, and suitable for the Artifacts tab.
